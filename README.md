# FolderQuick

一款 macOS 侧边文件夹管理窗口软件。

它的目标不是替代 Finder，而是做一个随叫随到的“文件工作台”：快速呼出、快速找文件、快速预览、快速拖拽到其他软件。

## 运行方式

进入 `app/FolderQuick` 后运行构建脚本：

```bash
./build_app.sh
```

构建完成后打开：

```text
app/FolderQuick/build/FolderQuick.app
```

如果系统不允许通过 `.app` 启动，可以直接运行：

```bash
./build/FolderQuick.app/Contents/MacOS/FolderQuick
```
