# FolderQuick

FolderQuick 是一款 macOS 侧边文件夹管理窗口软件。

它的目标不是替代 Finder，而是做一个随叫随到的“文件工作台”：把常用文件夹固定在一个轻量窗口里，需要文件时快速呼出、快速查找、快速预览、快速拖拽到其他软件。

## 产品介绍

FolderQuick 适合经常在多个项目、素材、下载、报销、归档目录之间来回切换的人。它把 Finder 里常用但分散的文件夹，整理成顶部标签页；点击菜单栏图标、按快捷键，或把鼠标移到屏幕边缘，就能立刻打开侧边窗口。

核心体验可以理解成“浏览器标签页 + Finder 文件区 + 侧边抽屉”：

- 常用文件夹用横排标签管理，像浏览器一样切换。
- 每个标签会记住自己停留在哪一层目录，来回切换不会丢失上下文。
- 文件区支持图标视图和列表视图，既能看缩略图，也能看紧凑列表。
- 图标和列表里都可以双击文件夹进入下一级。
- 底部层级小点可以快速回到前面某一级目录。
- 文件可拖入 FolderQuick，也可从 FolderQuick 拖到微信、邮件、浏览器、设计软件等其他 App。
- 拖入文件时，可以直接拖到某个文件夹上放手，文件会进入该文件夹；悬停在文件夹上还会自动进入下一层，方便一路拖到深层目录。
- 空格键支持 Quick Look 预览，复制、粘贴、重命名、删除等快捷键尽量贴近 Finder 习惯。
- 右键菜单提供在访达中显示、打开方式、重命名、复制路径、获取信息、删除到废纸篓、快速操作等常用动作。
- 设置里可调整窗口弹出位置、边缘触发、透明度、图标大小、图标间距、路径显示等。

## 运行方式

现在有两种构建方式。

推荐使用 Xcode 构建：

```bash
cd /Users/xiao-mbp2023/CodeSpace/Folder_Quick/app/FolderQuick
./build_xcode.sh
```

构建完成后应用在：

```text
app/FolderQuick/build/XcodeDerived/Build/Products/Debug/FolderQuick.app
```

也可以继续使用命令行 Swift 构建：

```bash
./build_app.sh
```

构建完成后应用在：

```text
app/FolderQuick/build/FolderQuick.app
```

如果系统不允许通过 `.app` 启动，可以直接运行：

```bash
./build/FolderQuick.app/Contents/MacOS/FolderQuick
```
