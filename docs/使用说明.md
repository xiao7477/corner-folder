# FolderQuick 使用说明

## 打开和呼出

- 点击菜单栏 FolderQuick 图标：显示或隐藏主窗口。
- 右键菜单栏图标：显示、隐藏或退出。
- 快捷键 `⌥⌘F`：显示或隐藏主窗口。
- 鼠标移动到设置中启用的屏幕边缘或角落：自动呼出窗口。
- 点右上角图钉：切换窗口置顶状态。

## 管理常用文件夹

- 顶部标签页代表常用文件夹。
- 点击标签页：切换到对应文件夹，并保留该标签上次停留的目录层级。
- 右键标签页：可在访达中打开、删除、回到首页、进入移动模式。
- 长按标签页：临时进入移动状态，拖动可调整顺序，松开结束移动。
- 新增文件夹后，标签栏会自动滚动到新标签可见的位置。

## 浏览文件

- 图标视图：适合看图片、视频、PDF 等缩略图。
- 列表视图：适合看大量文件，并可显示种类、修改日期、创建日期、大小等信息。
- 双击文件夹：进入该文件夹。
- 点击返回按钮：回到上一级。
- 底部小点：表示当前进入了几级目录，点击可快速回到对应层级。
- 左侧侧栏：显示当前目录下的文件夹树，单击文件夹即可跳转。

## 搜索、筛选和排序

- 搜索框：按文件名搜索。
- 空白处右键菜单：可打开时间筛选、文件类型筛选、排列方式、视图模式等。
- 列表顶部分类栏：左键点击可按对应字段排序，再点一次切换升序或降序。
- 置顶文件会排在当前页面最前面，不受普通排序影响。

## 文件操作

- 单击：选中文件。
- 双击：打开文件或进入文件夹。
- 空格：快速预览。
- 回车：在原文件名位置重命名。
- `⌘C`：复制选中文件。
- `⌘V`：把剪贴板里的文件粘贴到当前目录。
- `⌘⌫`：删除到废纸篓。
- 右键文件：在访达中显示、打开方式、重命名、复制路径、获取信息、删除到废纸篓、快速操作、置顶文件。

## 拖拽

- 从 FolderQuick 拖出文件：可拖到微信、邮件、浏览器、设计软件等其他 App。
- 从外部拖入文件：默认复制到当前目录。
- 按住 Option 拖入：移动到当前目录。
- 拖到某个文件夹上放手：复制或移动到该文件夹。
- 拖着文件悬停在文件夹、标签页或返回按钮上 1 秒：自动跳转，方便进入更深目录或退回上级。
- 拖拽过程中右键点击：取消当前拖拽。

## 设置

设置中可调整：

- 点击菜单栏图标后的窗口弹出位置。
- 是否启用边缘触发。
- 边缘触发位置和触发条显示。
- 置顶窗口是否允许被边缘触发隐藏。
- 窗口透明度。
- 图标大小和图标间距。
- 自动隐藏延迟。
- 是否显示底部路径。
- 是否显示左侧侧栏。
- 列表模式显示哪些信息列。

设置采用手动保存：修改后点击“保存设置”才会应用。
