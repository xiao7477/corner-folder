## 源码说明

- `main.swift`：应用入口、菜单栏图标、主控制器启动。
- `SidebarWindowController.swift`：主窗口、标签页、文件区、设置应用、拖拽和快捷操作。
- `SettingsPanel.swift`：设置窗口。
- `Models.swift`：文件模型、设置模型、日志工具。
- `FolderStore.swift`：常用文件夹和设置保存。
- `FileLoader.swift`：文件读取、类型判断、文件信息。
- `FileCollectionView.swift`、`FileItemView.swift`：图标视图。
- `FileOutlineView.swift`、`FileListRowView.swift`：列表视图。
- `FileDropScrollView.swift`、`FilePasteboardReader.swift`：拖入拖出文件。
- `FileTransferProgressPanel.swift`：文件复制或移动进度。
- `EdgeTriggerView.swift`：边缘触发区域。
- `GlobalHotkey.swift`：全局快捷键。

## 日志

运行时日志写入：

```text
~/Library/Application Support/FolderQuick/FolderQuick.log
```

如果出现卡死、设置不生效、拖拽异常，优先看这份日志。日志里会记录启动、读取文件、筛选排序、设置保存、列表列切换、边缘触发等关键节点。

## 当前技术路线

- 语言：Swift。
- 界面：AppKit。
- 文件预览：Quick Look。
- 缩略图：QuickLookThumbnailing。
- 文件拖拽：macOS Pasteboard 和 dragging API。
- 设置保存：本地 JSON 和 UserDefaults。
