# FolderQuick MVP 使用说明

## 产品定位

FolderQuick 是一个放在 macOS 菜单栏里的侧边文件工作台。它不是要替代访达，而是把你最常用的文件夹集中到一个可快速呼出的窗口里，让找文件、预览文件、拖拽文件这些高频动作更快完成。

## 当前已完成

- 菜单栏文件夹图标。
- 侧边悬浮窗口。
- 鼠标碰到屏幕右侧边缘时呼出窗口。
- 全局快捷键 `⌥⌘F` 显示或隐藏窗口。
- 菜单栏图标左键显示或隐藏窗口。
- 菜单栏图标右键打开菜单，可显示/隐藏或退出。
- 添加常用文件夹。
- 浏览器式横排标签页，点击标签可切换常用文件夹。
- 右键标签页可在访达中打开或删除标签。
- 拖动标签页可调整顺序。
- 显示当前文件夹路径。
- 返回收藏文件夹根目录。
- 刷新当前文件夹。
- 网格显示文件和文件夹。
- 支持图标视图和列表视图。
- 列表视图里文件夹可原地展开。
- 列表视图里双击文件夹可进入该文件夹层级。
- 图片、PDF、视频等文件缩略图。
- 文件名搜索。
- 文件类型筛选。
- 时间范围筛选。
- 双击打开文件。
- 双击文件夹进入下一级。
- 返回上一级文件夹。
- 空格键 Quick Look 快速预览。
- 文件拖拽到其他软件。
- 外部文件可拖入当前目录。
- 外部文件可直接拖到某个文件夹里。
- 拖拽悬停在文件夹上时，文件夹会闪烁并自动进入下一级，方便继续拖到更深层目录。
- 文件右键菜单支持：在访达中显示、打开方式、重命名、复制路径、获取信息、删除到废纸篓、快速操作。
- 右键打开或在 Finder 中显示。
- 窗口固定按钮。
- 基础设置窗口。

## 如何运行

推荐在项目的 `app/FolderQuick` 目录里运行 Xcode 构建脚本：

```text
./build_xcode.sh
```

会生成：

```text
app/FolderQuick/build/XcodeDerived/Build/Products/Debug/FolderQuick.app
```

也可以运行旧的命令行构建脚本：

```text
app/FolderQuick/build/FolderQuick.app
```

生成后会得到一个 `.app` 应用包。当前工具环境无法代替你双击打开自制 `.app`，但应用本体已经可以直接运行。你可以在 Finder 里双击尝试打开，也可以运行内部程序。

## 如何使用

1. 打开 FolderQuick。
2. 点击顶部的添加文件夹按钮。
3. 选择一个常用文件夹，例如“报销”“下载”“素材”。
4. 文件会以网格方式显示。
5. 可以搜索文件名，也可以按类型和时间筛选，输入时会即时刷新。
6. 双击文件可以打开。
7. 双击文件夹可以进入下一级。
8. 可以把文件直接拖到微信、邮件、浏览器或其他软件。
9. 也可以把外部文件拖进 FolderQuick；放在空白处会进入当前目录，放在某个文件夹上会进入那个文件夹。
10. 拖着文件停在文件夹上，FolderQuick 会自动进入下一层目录。
11. 按 `⌥⌘F` 可以显示或隐藏窗口。
12. 鼠标碰到屏幕右侧边缘，也可以呼出窗口。
13. 添加多个常用文件夹后，顶部会显示多个标签，点击标签即可切换。
14. 菜单栏文件夹图标：左键直接显示/隐藏，右键显示菜单。
15. 选中文件后按空格键，可以用系统 Quick Look 预览。
16. 列表视图里点击文件夹左侧箭头，可以展开下级文件。
17. 底部刷新按钮可以重新读取当前文件夹。

## 快捷键

- `⌘C`：复制选中文件。
- `⌘V`：把剪贴板里的文件粘贴到当前目录。
- 空格：快速预览。
- 回车：重命名。
- `⌘⌫`：删除到废纸篓。

## 当前限制

- 这是第一阶段 MVP，还不是正式发布版。
- 已经接入 Xcode 工程，可生成标准 `.app`。
- 正式 `.icns` 应用图标仍待完善。
- 快捷键暂时固定为 `⌥⌘F`，还不能在设置页里修改。
- 点击菜单栏图标的弹出位置支持上下左右和四个角。
- 屏幕边缘触发支持上下左右和四个角，并可多选。
- 多屏幕下会为每个屏幕生成对应触发条，鼠标在哪个屏幕触发，就在对应屏幕弹出。
- 快速操作目前先做了可控的简化版：压缩为 ZIP。Finder 原生完整快速操作和第三方扩展菜单暂未接入。
