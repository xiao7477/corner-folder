# FolderQuick MVP 使用说明

## 当前已完成

- 菜单栏文件夹图标。
- 侧边悬浮窗口。
- 鼠标碰到屏幕右侧边缘时呼出窗口。
- 全局快捷键 `⌥⌘F` 显示或隐藏窗口。
- 菜单栏图标左键显示或隐藏窗口。
- 菜单栏图标右键打开菜单，可显示/隐藏或退出。
- 添加常用文件夹。
- 切换常用文件夹。
- 顶部多标签页，点击标签可切换常用文件夹。
- 显示当前文件夹路径。
- 返回收藏文件夹根目录。
- 刷新当前文件夹。
- 网格显示文件和文件夹。
- 图片、PDF、视频等文件缩略图。
- 文件名搜索。
- 文件类型筛选。
- 时间范围筛选。
- 双击打开文件。
- 双击文件夹进入下一级。
- 返回上一级文件夹。
- 空格键 Quick Look 快速预览。
- 文件拖拽到其他软件。
- 右键打开或在 Finder 中显示。
- 窗口固定按钮。
- 基础设置窗口。

## 如何运行

推荐在项目的 `app/FolderQuick` 目录里运行 Xcode 构建脚本：

```text
./build_xcode.sh
```

会生成：

```text
app/FolderQuick/build/XcodeDerived/Build/Products/Debug/FolderQuick.app
```

也可以运行旧的命令行构建脚本：

```text
app/FolderQuick/build/FolderQuick.app
```

生成后会得到一个 `.app` 应用包。当前工具环境无法代替你双击打开自制 `.app`，但应用本体已经可以直接运行。你可以在 Finder 里双击尝试打开，也可以运行内部程序。

## 如何使用

1. 打开 FolderQuick。
2. 点击顶部的添加文件夹按钮。
3. 选择一个常用文件夹，例如“报销”“下载”“素材”。
4. 文件会以网格方式显示。
5. 可以搜索文件名，也可以按类型和时间筛选，输入时会即时刷新。
6. 双击文件可以打开。
7. 双击文件夹可以进入下一级。
8. 可以把文件直接拖到微信、邮件、浏览器或其他软件。
9. 按 `⌥⌘F` 可以显示或隐藏窗口。
10. 鼠标碰到屏幕右侧边缘，也可以呼出窗口。
11. 添加多个常用文件夹后，顶部会显示多个标签，点击标签即可切换。
12. 菜单栏文件夹图标：左键直接显示/隐藏，右键显示菜单。
13. 选中文件后按空格键，可以用系统 Quick Look 预览。
14. 点击顶部的小房子按钮，可以回到当前收藏文件夹根目录。
15. 点击刷新按钮，可以重新读取当前文件夹。

## 当前限制

- 这是第一阶段 MVP，还不是正式发布版。
- 暂时没有完整 Xcode 工程。
- 构建脚本会生成 1024 PNG 图标源；当前命令行环境下 `iconutil` 没能生成 `.icns`。
- 快捷键暂时固定为 `⌥⌘F`，还不能在设置页里修改。
- 屏幕边缘触发支持左侧或右侧，但触发条样式仍然比较朴素。

## 下一步建议

- 增加快捷键自定义。
- 完成正式 `.icns` 应用图标。
- 增加标签、置顶和批量操作。
- 用完整 Xcode 工程做签名和打包。
