# 更新记录

## 0.5.0

- 增加标签页状态记忆：切换标签只切换工作区，不再强制回到该标签首页。
- 标签页右键新增“回到首页”，需要回到收藏根目录时可手动触发。
- 图标视图和列表视图都支持底部层级小点，可快速回到上级路径节点。
- 底部路径过长时会自动中间省略，鼠标悬停可查看完整路径。
- 列表视图双击文件夹会进入该文件夹层级。
- 拖入文件时支持放到指定文件夹，不再只能放到当前目录。
- 拖拽文件悬停在文件夹上时，文件夹会闪烁并自动进入下一级，便于继续拖到更深层目录。
- 保留同名文件处理弹窗：替换、保留两者或跳过。

## 0.2.0

- 增加当前路径显示。
- 增加返回收藏文件夹根目录。
- 增加刷新当前文件夹。
- 增加空格键 Quick Look 预览。
- 增加可用设置页：窗口位置、透明度、图标大小、自动隐藏延迟、边缘触发条显示。
- 记住窗口大小和设置。
- 搜索框输入时即时刷新。
- 图标大小设置会影响网格尺寸和缩略图尺寸。
- 构建脚本会生成应用图标 PNG 源，并在系统允许时生成 `.icns`。

## 0.3.0

- 新增标准 Xcode 工程 `FolderQuick.xcodeproj`。
- 新增 Xcode 构建脚本 `build_xcode.sh`。
- Xcode Debug 构建会输出标准 `.app` 到 `build/XcodeDerived/Build/Products/Debug/FolderQuick.app`。
- 保留原来的命令行 Swift 构建脚本，方便快速验证。

## 0.4.0

- 顶部改为浏览器式横排标签页。
- 标签页支持右键在访达打开、删除标签。
- 标签页支持拖拽排序。
- 刷新按钮下沉到底部设置旁边。
- 底部路径可在设置中开关。
- 点击菜单栏图标的弹出位置扩展为上下左右和四个角。
- 边缘触发条支持上下左右和四个角多选，并支持多屏幕对应触发。
- 文件右键菜单保留：在访达中显示、打开方式、重命名、复制路径、获取信息、删除到废纸篓、快速操作。
- 快捷键对齐 Finder 常用习惯：复制、粘贴、预览、删除、重命名。
- 新增列表视图，文件夹支持原地展开。
- 修复标签页点击区域偏移、右键不稳定、滚轮方向不符合预期的问题。
- 修复图标模式右键和双击打开不稳定的问题。
- 修复设置窗口在多显示器下打开位置不符合当前屏幕的问题。
- 设置中新增启用边缘触发总开关。
- 设置中新增图标间距，并将图标大小范围扩展为 56 到 168。
- 图标模式缩略图现在会随图标大小同步缩放。

当前已知限制：

- 快速操作目前是简化版，仅提供压缩为 ZIP。
- 获取信息是自定义信息窗口，不是 Finder 原生“显示简介”窗口。
- 正式 `.icns` 图标打包仍待完善。

## 0.1.0

- 完成侧边悬浮窗口。
- 支持添加和切换常用文件夹。
- 支持顶部多标签。
- 支持网格浏览文件。
- 支持搜索、类型筛选、时间筛选。
- 支持拖拽文件。
- 支持菜单栏图标左键显示/隐藏、右键菜单。
- 支持 `⌥⌘F` 快捷键。
